# indus.io

## Description

A B2B platform for simulating, managing and optimising industrial production lines.

## Run

### Prerequisites
- Docker Desktop

### Steps

1. Clone the repo

2. Create your environment file from the example shown

3. Generate a secret key and paste it in .env

4. Start the services <br />
`docker compose up --build`

- Frontend: http://localhost:5173
- Backend API: http://localhost:8000
- API docs: http://localhost:8000/docs
